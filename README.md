# Student Performance Analysis
> A data analysis project exploring factors affecting student performance

## Overview

This project explores how different factors influence student performance using a real world dataset. 
The aim was to understand patterns in scores and identify which aspects actually influence student performance.

## Approach 

* Cleaned and explored the dataset using Pandas
* Analyzed relationships between student scores and background factors
* Created visualizations to make patterns easy to understand

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Key Observations

* Students with higher parental education tend to perform better
* Parental marital status does not show a strong impact on scores
* Mathematics appears to be the most challenging subject for most students
* The distribution of students across ethnic groups is fairly varied

## Visualizations

* Bar charts for comparing groups
* Pie charts to understand distributions (including ethnic groups)
* Box plots to analyze score spread across subjects

## Conclusion

This analysis highlights that not all background factors affect performance equally. While some variables show clear influence, others have little to no effect. 
It also shows that mathematics remains a challenging subject for many students, indicating a need for focused improvement.
